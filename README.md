### Project Description
GenoGuard AI is an intelligent, web-based diagnostic assistance tool that helps detect the risk of genetic disorders in unborn babies during pregnancy. It analyzes biochemical blood/urine markers (HCG, PAPP-A, AFP, Urine Protein) and fetal ultrasound images using AI/ML models to provide an early, non-invasive risk score.

### The Problem statement
Thousands of babies are born each year with genetic disorders that could have been detected earlier. Traditional tests like amniocentesis are invasive and risky. GenoGuard AI offers a fast, non-invasive, web-based screening alternative.

### The Solution
GenoGuard AI solves the problem of risky and invasive prenatal diagnostic procedures by providing an AI-powered, non-invasive web-based screening tool. It analyzes biochemical markers and ultrasound data to generate an early risk score for genetic disorders, helping doctors make faster and safer clinical decisions

## Features

List the key features of your project:
- Feature 1: GenoGuard AI predicts genetic disorder risk using biochemical blood and urine markers, eliminating the need for invasive procedures like Amniocentesis.
- Feature 2: The system uses Machine Learning algorithms to analyze HCG, PAPP-A, AFP, and urine protein levels and generates a numerical risk score (0–100) with Low, Medium, or High classification.
- Feature 3: It is a browser-based application (HTML/CSS frontend + FastAPI backend) that allows doctors to access the tool from anywhere without installing special software.
- Feature 4: The system processes patient data instantly and provides immediate diagnostic findings, supporting faster medical decision-making.
